# romanczuk.vercel.app — Personal Portfolio

Personal portfolio and CV site for Piotr Romanczuk, Software Engineer based in Warsaw, Poland.

**Live:** [romanczuk.vercel.app](https://romanczuk.vercel.app)

## Tech Stack

- **Framework:** Next.js 16 (App Router, React 19)
- **Styling:** Tailwind CSS 4, shadcn/ui (Radix UI)
- **Animations:** Motion (Framer Motion)
- **Fonts:** Space Grotesk + Space Mono
- **Testing:** Playwright E2E
- **Deployment:** Vercel

## Features

- Responsive landing page with hero, about, projects, timeline, and contact sections
- Dark/light theme toggle
- Terminal-style project cards with animated output
- Contact form (Formspree integration)
- Dedicated `/cv` route with print-optimized A4 layout (Ctrl+P to save as PDF)
- Mobile-first design with smooth scroll navigation

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
